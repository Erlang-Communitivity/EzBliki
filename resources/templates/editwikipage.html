About this page

This page allows you to edit an existing Wiki entry, in which case the fields already contain the existing entry property values. It also allows you to create a new wiki entry, in which case the fields are mostly blank. When you're done click the Update/Create button, or click cancel if you want to stop and go back to the home page.
